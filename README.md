# Michal Maciej Mazur — portfolio site

This folder is a self-contained static website ready for a GitHub repository and GitHub Pages.

## Publish with GitHub Pages

1. Create a new GitHub repository.
2. Upload the **contents** of this folder (not the `github-site` folder itself) to the repository root.
3. In GitHub, open **Settings → Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Select the `main` branch and the `/ (root)` folder, then save.

GitHub will publish `index.html` as the homepage. A deployment URL will appear on the same settings screen.
